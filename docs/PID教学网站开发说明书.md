# PID 参数交互式教学网站 · 开发说明书

## 1. 项目概述

**项目名称**：PID 交互式教学实验室（PID Interactive Lab）

**目标**：制作一个无需安装、双击即可在常规电脑浏览器中打开的交互式网页，
通过实时动画 + 可调参数 + 多维度讲解，帮助学习者直观理解 PID 控制器中
Kp（比例）、Ki（积分）、Kd（微分）三个参数的作用。

**目标用户**：自动化/电子/机器人专业学生、嵌入式工程师、创客爱好者。

## 2. 技术选型

| 项目 | 选择 | 理由 |
|---|---|---|
| 形态 | 单文件 `pid-tutorial.html` | 双击即开，无需服务器、无需安装 |
| 渲染 | 原生 Canvas 2D | 零依赖，离线可用，性能足够 |
| 逻辑 | 原生 JavaScript（ES6） | 无构建步骤，任意现代浏览器可运行 |
| 样式 | 内嵌 CSS，深色主题 | 单文件自包含 |
| 兼容 | Chrome / Edge / Firefox / Safari 近 5 年版本 | 常规电脑全覆盖 |

**明确不使用**：任何 CDN、外部字体、外部图表库（保证断网环境可用）。

## 3. 功能模块

页面采用顶部 Tab 导航，共 5 个页签：

### 3.1 原理讲解页
从多个维度解析 PID：
- **数学维度**：连续/离散公式，逐项拆解 P、I、D 的表达式含义；
- **物理直觉维度**：以"开车踩油门"类比（P=看差距踩油门，I=记仇补偿，D=预判刹车）；
- **时域响应维度**：上升时间、超调量、稳态误差、调节时间与三参数的关系表；
- **工程实践维度**：积分饱和(windup)与抗饱和、微分对噪声敏感、输出限幅、采样周期。
- 附"试凑法调参口诀"与调参步骤。

### 3.2 参数实验室页（通用对象）
- 被控对象：带阻尼的二阶系统（位置控制），最能展现振荡/超调现象；
- 三个滑块实时调节 Kp、Ki、Kd，曲线即时刷新；
- 预设按钮：纯 P / PI / PD / 完整 PID / 激进 / 保守；
- 实时曲线：设定值（虚线）、实际值（实线）、控制输出；
- P/I/D 三个分量的实时贡献条形图；
- 指标面板：当前误差、最大超调量、是否进入 ±2% 带。

### 3.3 场景一：电机转速控制
- **物理模型**：`J·ω' = K·u − b·ω − T_load`（J=0.1, b=0.05, K=0.5，电压限幅 ±24V）；
- **动画**：旋转圆盘 + 指针转速表，转速与仿真值同步；
- **扰动**：一键"加载/卸载负载"，演示纯 P 控制掉速、加 I 后恢复；
- **讲解重点**：为什么带负载时必须有积分项消除稳态转速差。

### 3.4 场景二：恒温箱温度控制
- **物理模型**：`C·T' = P_max·u − h·(T − T_env)`（C=500 J/K, h=10 W/K, P_max=1000W，u∈[0,1] 只能加热）；
- 时间加速 20 倍播放（热过程慢）；
- **动画**：烧杯液体颜色随温度由蓝变红，加热器辉光随输出功率变化；
- **扰动**："开盖散热"（散热系数临时×3）、"倒入冷水"（温度突降）；
- **讲解重点**：单向执行器 + 大滞后系统中积分饱和的危害与抗饱和；D 项抑制超调。

### 3.5 场景三：无人机定高悬停
- **物理模型**：`m·z'' = u·F_max − m·g`（m=1kg, F_max=20N, u∈[0,1]）；
- **动画**：侧视四旋翼上下运动，桨叶模糊度随油门变化，目标高度虚线；
- **扰动**："阵风"（瞬时向下速度冲击）；
- **讲解重点**：重力是恒定扰动 → 必须 I 项（或前馈）才能悬停；D 项等效空气阻尼防振荡。

## 4. 仿真引擎设计

- **定步长积分**：欧拉法，dt = 2ms（温度场景内部按时间加速倍数多步推进）；
- **渲染**：requestAnimationFrame，每帧推进若干仿真步，只运行当前激活页签；
- **PID 实现**：
  - 位置式 PID；首拍跳过微分（避免无意义尖峰）；
  - 输出限幅 + 反算抗积分饱和（back-calculation）；
  - 每次改参数/重置时清空积分与历史误差；
- **历史数据**：环形窗口（实验室/电机/无人机约 12s，温度约 300s 仿真时间）。

## 5. UI / 交互规范

- 深色主题，主色 #4fc3f7；中文界面；
- 每个仿真页布局：左侧动画 Canvas（约 300px 宽）+ 右侧曲线 Canvas（自适应），
  下方参数滑块区 + 预设/扰动按钮 + 指标区 + 该场景的文字讲解卡片；
- 滑块旁实时显示数值；所有按钮即点即生效；
- 窗口缩放时 Canvas 自适应重绘。

## 6. 验收标准

1. 在无网络环境下双击 HTML 文件可正常打开并运行所有页签；
2. 拖动任一滑块，曲线与动画在 100ms 内反映新参数；
3. 实验室页可复现：纯 P 有稳态误差、加 I 消除、Ki 过大振荡、加 D 抑制超调；
4. 电机页加负载后：纯 P 掉速不恢复，PI 可恢复设定转速；
5. 温度页可观察积分饱和导致的大超调现象；
6. 无人机页 Ki=0 时悬停高度低于目标（重力稳态误差），加 I 后贴线悬停；
7. 单文件体积 < 200KB，普通笔记本 60fps 流畅运行。
